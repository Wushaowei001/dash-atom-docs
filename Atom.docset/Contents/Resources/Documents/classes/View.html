    Class View

    Defined in: docs/includes/space-pen/src/space-pen.coffee
    Inherits: jQuery

    Overview

    View class that extends the jQuery prototype.

    Extending classes must implement a @content method.

    Example

    class Spacecraft extends View
  @content: ->
    @div =>
      @h1 'Spacecraft'
      @ol =>
        @li 'Apollo'
        @li 'Soyuz'
        @li 'Space Shuttle'

    Each view instance will have all the methods from the jQuery prototype available on it.

      craft = new Spacecraft()
  craft.find('h1').text() # 'Spacecraft'
  craft.appendTo(document.body) # View is now a child of the <body> tag

    ::command(commandName, selector, options, handler) Source

    Calls the given handler when commandName is triggered on the View.

    This is enhanced version of jQuery's ::on method. It listens for a custom DOM event and adds metadata to the DOM to maintain a list of all commands.

    Parameters:

    • (String) commandName — A namespaced String describing the command, such as find-and-replace:toggle.
    • (String) selector — An optional selector String to filter the descendants of the elements that trigger the event.
    • (Object) options — An optional options Object with an data key.
    • (Function) handler — A Function to execute when the command is triggered.
