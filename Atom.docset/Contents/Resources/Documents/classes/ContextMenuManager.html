    Class ContextMenuManager

    Defined in: src/context-menu-manager.coffee

    Overview

    Provides a registry for commands that you'd like to appear in the context menu.

    An instance of this class is always available as the atom.contextMenu global.

    Instance Method Summary

    • ::add(name, object, {}={}) Creates menu definitions from the object specified by the menu cson API.
    • ::showForEvent(event) Request a context menu to be displayed.

    Instance Method Details

    ::add(name, object, {}={}) Source

    Creates menu definitions from the object specified by the menu cson API.

    Parameters:

    • name — The path of the file that contains the menu definitions.
    • object — The 'context-menu' object specified in the menu cson API.
    • (Object) options — An Object with the following keys:
      • devMode — Determines whether the entries should only be shown when the window is in dev mode.

    Returns:

    • Returns nothing.
