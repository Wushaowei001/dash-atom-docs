    Class Grammar

    Defined in: docs/includes/first-mate/src/grammar.coffee

    Overview

    Grammar that tokenizes lines of text.

    This class should not be instantiated directly but instead obtained from a GrammarRegistry by calling GrammarRegistry::loadGrammar.

    Instance Method Summary

    • Object ::tokenizeLine(line, ruleStack, firstLine = false) Tokenize the line of text.
    • Array ::tokenizeLines(text) Tokenize all lines in the given text.

    Instance Method Details

    Object ::tokenizeLine(line, ruleStack, firstLine = false) Source

    Tokenize the line of text.

    Parameters:

    • (String) line — A String of text to tokenize.
    • (Array) ruleStack — An optional Array of rules previously returned from this method. This should be null when tokenizing the first line in the file.
    • (Boolean) firstLine — A Boolean denoting whether this is the first line in the file which defaults to false. This should be true when tokenizing the first line in the file.

    Returns:

    • (Object) — Returns an Object containing tokens and ruleStack properties:
      • (Array) token — An Array of tokens covering the entire line of text.
      • (Array) ruleStack — An Array of rules representing the tokenized state at the end of the line. These should be passed back into this method when tokenizing the next line in the file.

    Array ::tokenizeLines(text) Source

    Tokenize all lines in the given text.

    Parameters:

    • (String) text — A String containing one or more lines.

    Returns:

    • (Array) — Returns an Array of token arrays for each line tokenized.
