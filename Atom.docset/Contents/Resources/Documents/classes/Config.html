    Class Config

    Defined in: src/config.coffee

    Overview

    Used to access all of Atom's configuration details.

    An instance of this class is always available as the atom.config global.

    Best practices

    • Create your own root keypath using your package's name.
    • Don't depend on (or write to) configuration keys outside of your keypath.

    Example

    atom.config.set('my-package.key', 'value')
atom.config.observe 'my-package.key', ->
  console.log 'My configuration changed:', atom.config.get('my-package.key')

    Object ::observe(keyPath, options = {}, callback) Source

    Establishes an event listener for a given key.

    callback is fired whenever the value of the key is changed and will be fired immediately unless the callNow option is false.

    Parameters:

    • (String) keyPath — The String name of the key to observe
    • (Object) options — An optional Object containing the callNow key.
    • (Function) callback — The Function to call when the value of the key changes. The first argument will be the new value of the key and the second argument will be an Object with a previous property
    • that is the prior value of the key.

    Returns:

    • (Object) — Returns an Object with the following keys:
      • (Function) off — A Function that unobserves the `keyPath` when called.
