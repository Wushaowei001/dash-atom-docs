    Class ScrollView

    Defined in: src/scroll-view.coffee
    Inherits: View

    Overview

    Represents a view that scrolls.

    Subclasses must call super if overriding the initialize method or else the following events won't be handled by the ScrollView.

    Events

    • core:move-up
    • core:move-down
    • core:page-up
    • core:page-down
    • core:move-to-top
