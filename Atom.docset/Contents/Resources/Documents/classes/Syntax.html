    Class Syntax

    Defined in: src/syntax.coffee
    Inherits: GrammarRegistry

    Overview

    Syntax class holding the grammars used for tokenizing.

    An instance of this class is always available as the atom.syntax global.

    The Syntax class also contains properties for things such as the language-specific comment regexes.
