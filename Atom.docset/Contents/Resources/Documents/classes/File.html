    Class File

    Defined in: docs/includes/node-pathwatcher/src/file.coffee

    Overview

    Represents an individual file that can be watched, read from, and written to.

    Instance Method Summary

    • ::constructor(filePath, symlink = false) Creates a new file.
    • ::exists() Returns whether the file exists.
    • ::getBaseName() Return the String filename without any directory information.
    • ::getDigest() Get the SHA-1 digest of this file
    • ::getParent() Return the Directory that contains this file.
    • ::getPath() Returns the String path for the file.
    • ::getRealPathSync() Returns this file's completely resolved path.
    • ::isDirectory() Distinguishes Files from Directories during traversal.
    • ::isFile() Distinguishes Files from Directories during traversal.
    • ::read(flushCache) Reads the contents of the file.
    • ::write(text) Overwrites the file with the given String.
