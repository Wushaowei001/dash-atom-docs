    Class Workspace

    Defined in: src/workspace.coffee
    Inherits: Model

    Overview

    Represents the state of the user interface for the entire window. An instance of this class is available via the atom.workspace global.

    Interact with this object to open files, be notified of current and future editors, and manipulate panes. To add panels, you'll need to use the WorkspaceView class for now until we establish APIs at the model layer.

    Editor ::open(uri, options = {}) Source

    Open a given a URI in Atom asynchronously.

    Parameters:

    • (String) uri — A String containing a URI.
    • (Object) options — An optional options Object
      • (Number) initialLine — A Number indicating which row to move the cursor to initially. Defaults to 0.
      • (Number) initialColumn — A Number indicating which column to move the cursor to initially. Defaults to 0.
      • split — Either 'left' or 'right'. If 'left', the item will be opened in leftmost pane of the current active pane's row. If 'right', the item will be opened in the rightmost pane of the current active pane's row.
      • (Boolean) activatePane — A Boolean indicating whether to call Pane::activate on the containing pane. Defaults to true.
      • (Boolean) searchAllPanes — A Boolean. If true, the workspace will attempt to activate an existing item for the given URI on any pane. If false, only the active pane will be searched for an existing item for the same URI. Defaults to false.

    Returns:

    • (Editor) — Returns a promise that resolves to the Editor for the file URI.

    Register an opener for a uri.

    An Editor will be used if no openers return a value.

    Example

  if path.extname(uri) is '.toml'
    return new TomlEditor(uri)

    ::saveActivePaneItem() Source

    Save the active pane item.

    If the active pane item currently has a URI according to the item's .getUri method, calls .save on the item. Otherwise ::saveActivePaneItemAs # will be called instead. This method does nothing if the active item does not implement a .save method.

    ::saveActivePaneItemAs() Source

    Prompt the user for a path and save the active pane item to it.

    Opens a native dialog where the user selects a path on disk, then calls .saveAs on the item with the selected path. This method does nothing if the active item does not implement a .saveAs method.
