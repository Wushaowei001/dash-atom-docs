    Class PaneView

    Defined in: src/pane-view.coffee
    Inherits: View

    Overview

    A container which can contains multiple items to be switched between.

    Items can be almost anything however most commonly they're EditorViews.

    Most packages won't need to use this class, unless you're interested in building a package that deals with switching between panes or items.
