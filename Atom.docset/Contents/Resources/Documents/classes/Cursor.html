    Class Cursor

    Defined in: src/cursor.coffee
    Inherits: Model

    Overview

    The Cursor class represents the little blinking line identifying where text can be inserted.

    Cursors belong to Editors and have some metadata attached in the form of a Marker.

    Range ::getBeginningOfCurrentWordBufferPosition(options = {}) Source

    Retrieves the buffer position of where the current word starts.

    Parameters:

    • (Object) options — An Object with the following keys:
      • (RegExp) wordRegex — A RegExp indicating what constitutes a "word" (default: ::wordRegExp).
      • (Boolean) includeNonWordCharacters — A Boolean indicating whether to include non-word characters in the default word regex. Has no effect if wordRegex is set.
      • (Boolean) allowPrevious — A Boolean indicating whether the beginning of the previous word can be returned.
