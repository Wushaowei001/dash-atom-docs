    Class MenuManager

    Defined in: src/menu-manager.coffee

    Overview

    Provides a registry for menu items that you'd like to appear in the application menu.

    An instance of this class is always available as the atom.menu global.

    Instance Method Summary

    • ::add(items) Adds the given item definition to the existing template.
    • ::update() Refreshes the currently visible menu.

    Instance Method Details

    ::add(items) Source

    Adds the given item definition to the existing template.

    Example

    atom.menu.add [
  {
    label: 'Hello'
    submenu : [{label: 'World!', command: 'hello:world'}]
  }
]

    Parameters:

    • (Array) items — An Array of menu item Objects containing the keys:
      • (String) label — The String menu label.
      • (Array) submenu — An optional Array of sub menu items.
      • (String) command — An optional String command to trigger when the item is clicked.

    Returns:

    • Returns nothing.

    Refreshes the currently visible menu.
