    Class Atom

    Defined in: src/atom.coffee
    Inherits: Model

    Overview

    Atom global for dealing with packages, themes, menus, and the window.

    An instance of this class is always available as the atom global.

    Useful properties available:

    • atom.clipboard - A Clipboard instance
    • atom.config - A Config instance
    • atom.contextMenu - A ContextMenuManager instance
    • atom.deserializers - A DeserializerManager instance
    • atom.keymaps - A KeymapManager instance
    • atom.menu - A MenuManager instance
    • atom.packages - A PackageManager instance
    • atom.project - A Project instance
    • atom.syntax - A Syntax instance
    • atom.themes - A ThemeManager instance
    • atom.workspace - A Workspace instance
    • atom.workspaceView - A WorkspaceView instance

    Number ::confirm({message, detailedMessage, buttons}={}) Source

    Open a confirm dialog.

    Example

    atom.confirm
  message: 'How you feeling?'
  detailedMessage: 'Be honest.'
  buttons:
    Good: -> window.alert('good to hear')
    Bad:  -> window.alert('bummer')

    Parameters:

    • (Object) options — An Object with the following keys:
      • (String) message — The String message to display.
      • (String) detailedMessage — The String detailed message to display.
      • buttons — Either an array of strings or an object where keys are button names and the values are callbacks to invoke when clicked.

    Returns:

    • (Number) — Returns the chosen button index Number if the buttons option was an array.

    ::setWindowDimensions({x, y, width, height}) Source

    Set the dimensions of the window.

    The window will be centered if either the x or y coordinate is not set in the dimensions parameter. If x or y are omitted the window will be centered. If height or width are omitted only the position will be changed.

    Parameters:

    • (Object) dimensions — An Object with the following keys:
      • x — The new x coordinate.
      • y — The new y coordinate.
      • width — The new width.
      • height — The new height.
