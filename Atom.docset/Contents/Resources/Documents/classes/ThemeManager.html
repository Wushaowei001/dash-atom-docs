    Class ThemeManager

    Defined in: src/theme-manager.coffee

    Overview

    Handles loading and activating available themes.

    An instance of this class is always available as the atom.themes global.

    Instance Method Summary

    • ::getActiveNames() Get an array of all the active theme names.
    • ::getActiveThemes() Get an array of all the active themes.
    • ::getLoadedNames() Get an array of all the loaded theme names.
    • ::getLoadedThemes() Get an array of all the loaded themes.
    • ::getUserStylesheetPath() Returns the String path to the user's stylesheet under ~/.atom
    • ::requireStylesheet(stylesheetPath, type = 'bundled', htmlElement) Resolve and apply the stylesheet specified by the path.
    • ::setEnabledThemes(enabledThemeNames) Set the list of enabled themes.
