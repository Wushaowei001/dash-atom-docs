    Class Clipboard

    Defined in: src/clipboard.coffee

    Overview

    Represents the clipboard used for copying and pasting in Atom.

    An instance of this class is always available as the atom.clipboard global.

    Instance Method Summary

    • String ::read() Read the text from the clipboard.
    • Object ::readWithMetadata() Read the text from the clipboard and return both the text and the associated metadata.
    • ::write(text, metadata) Write the given text to the clipboard.

    Object ::readWithMetadata() Source

    Read the text from the clipboard and return both the text and the associated metadata.

    Returns:

    • (Object) — Returns an Object with the following keys:
      • (String) text — The String clipboard text.
      • metadata — The metadata stored by an earlier call to ::write.

    ::write(text, metadata) Source

    Write the given text to the clipboard.

    The metadata associated with the text is available by calling ::readWithMetadata.

    Parameters:

    • (String) text — The String to store.
    • metadata — The additional info to associate with the text.
