    Overview

    Represents a directory on disk that can be watched for changes.

    Configures a new Directory instance, no files are accessed.

    • (Boolean) symlink — A Boolean indicating if the path is a symlink (default: false).

    Instance Method Details

    ::contains(pathToCheck) Source

    Returns whether the given path (real or symbolic) is inside this directory. This method does not actually check if the path exists, it just checks if the path is under this directory.

    ::getBaseName() Source

    Returns the String basename of the directory.

    ::getEntries(callback) Source

    Reads file entries in this directory from disk asynchronously.

    Parameters:

    • (Function) callback — A Function to call with an Error as the 1st argument and an Array of File and Directory objects as the 2nd argument.

    Array ::getEntriesSync() Source

    Reads file entries in this directory from disk synchronously.

    Returns:

    • (Array) — Returns an Array of File and Directory objects.

    ::getFile(filename...) Source

    Traverse within this Directory to a child File. This method doesn't actually check to see if the File exists, it just creates the File object.

    Parameters:

    • filename — The name of a File within this Directory.

    ::getParent() Source

    Traverse to the parent Directory.

    ::getPath() Source

    Returns the directory's symbolic path.

    This may include unfollowed symlinks or relative directory entries. Or it may be fully resolved, it depends on what you give it.

    ::getRealPathSync() Source

    Returns this directory's completely resolved path.

    All relative directory entries are removed and symlinks are resolved to their final destination.

    ::getSubdirectory(dirname...) Source

    Traverse within this a Directory to a child Directory. This method doesn't actually check to see if the Directory exists, it just creates the Directory object.

    Parameters:

    • dirname — The name of the child Directory.
